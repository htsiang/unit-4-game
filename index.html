Rupee Collector!

You will be given a random number of rupees at the start of the game.

There are four rupee types below. By clicking on a rupee type, you will add a specific amount of rupees to your total score.

You win the game by matching your total number of rupees to the random number of rupees. You lose the game if your total score goes above the random number.

The value of each rupee type is hidden from you until you click on it.

Each time the game starts, the value of each rupee type will change.
